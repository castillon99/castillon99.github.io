# Fundamentos de GitHub

Proyecto web independiente para aprender GitHub desde cero.

## Qué contiene

- `index.html`: página completa con diseño, contenido, estilos y JavaScript.
- Secciones sobre errores comunes, herramientas útiles, GitHub Pages, comandos base y práctica guiada.

## Cómo publicarlo sin tocar la portada principal

Sube la carpeta completa `fundamentos-de-github` dentro de tu repositorio `castillon99.github.io`.

La URL esperada será:

`https://castillon99.github.io/fundamentos-de-github/`

No reemplaces el `index.html` de la raíz del repositorio, porque ese archivo es tu portada principal.
